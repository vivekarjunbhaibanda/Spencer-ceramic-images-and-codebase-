# Spencer Ceramic - Images & Codebase

This repo is connected to **Lovable** for building the product section of the website.

## 🔗 How to Use in Lovable

Since this repository is already connected to Lovable:

1. In Lovable, go to your project.
2. Use the GitHub integration / Assets section.
3. Reference images using the path, for example:

```
images/wood-match-collection/Wood_Matt_Collection_page-0001.jpg
```

Or use the **raw GitHub URL** format:

```
https://raw.githubusercontent.com/vivekarjunbhaibanda/Spencer-ceramic-images-and-codebase-/main/images/wood-match-collection/filename.jpg
```
